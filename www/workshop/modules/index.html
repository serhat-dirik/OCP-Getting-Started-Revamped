OpenShift Application Platform — Getting Started

Welcome to the OpenShift Application Platform — Getting Started workshop.

The Parasol story

Parasol Insurance is modernizing how it processes claims. You join Parasol’s engineering organization, and across this workshop you take one application — the Parasol claims platform — from its first deployment to a governed, observable, AI-assisted service on OpenShift. Every module solves the next real problem Parasol hits, so each platform capability arrives as the answer to a question you already have.

How to use this workshop

This workshop is a library of 27 self-contained modules, not one track you must follow in order. Any module can be your first: you never need to have finished an earlier module to start. When you begin a module, its world — namespaces, applications, Git repositories, pipelines, and access — is materialized for you on demand:

ws start m01

Run ws verify m01 to check your environment, and ws reset m01 to wipe it and start clean. Pick one of the recommended paths below, or assemble your own — roughly four hands-on modules make a day.
