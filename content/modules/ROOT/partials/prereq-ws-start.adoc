[NOTE]
.Before you start — prepare your environment
====
This module is self-contained: you do *not* need to have completed any earlier module.
Prepare your environment from the *Terminal* tab — safe to run any time:

[source,sh,role=execute,subs="attributes"]
----
ws prep {module-id}
----

It checks your project, tells you if anything is left over from a previous module, and *asks before wiping* anything to set up this module's world.
If it says your environment is already prepared, you are good to go.
Only if `ws prep` itself reports a platform error, tell your instructor.
====
